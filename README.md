# Binly API

[http://binly.co.uk](binly.co.uk) is a web app that allows a users to check thier bins.
Created using typescript express api which interfaces with a postgres database.

## Packages

- axios
- express
- ts-node
- ts-postgres
- typescript

## Supported Councils

- Glasgow City Council
